# 中文写作技能指南

这份指南旨在指导大模型生成高质量、现代风格的简体中文内容。

## 像人一样写作

- 句子长度和结构前后有变化。
- 有清晰的观点和态度。使用中性语气能增加客观感。示例： “我们观察到...” 可改为 “实验观察结果表明...”。
- 直接陈述事实，避免绕圈和过度铺垫。

## 排版规则

- 中英文混排：英文/数字与中文之间保留一个空格（如`在 Windows 中上传 5 张图片`）
- 标点符号：使用全角标点（逗号，句号。冒号：左引号“ 右引号”）
- 避免行话：首次使用术语时用通俗语言解释

## 回避过于夸张的语言

**以下词汇需要谨慎使用：** 作为/充当、标志着、见证了、是……的体现/证明/提醒、极其重要的/重要的/至关重要的/核心的/关键性的作用/时刻、凸显/强调/彰显了其重要性/意义、反映了更广泛的、象征着其持续的/永恒的/持久的、为……做出贡献、为……奠定基础、标志着/塑造着、代表/标志着一个转变、关键转折点、不断演变的格局、焦点、不可磨灭的印记、深深植根于
